Google Sheets Integration

Connect directly to the Google Sheets API — no Apps Script deployment required.

📋 One-time setup (done by you, in Google Cloud)

Step 1. In console.cloud.google.com, create a project and enable the Google Sheets API.
Step 2. Under APIs & Services → Credentials, create an OAuth client ID (type: Web application) and add this page's URL as an Authorized JavaScript origin.
Step 3. Paste that Client ID below, plus the Spreadsheet ID from your sheet's URL.
Step 4. Click Connect Google Account and sign in — this authorizes the app for the current session only.

Connection Settings

You'll still need to click "Connect Google Account" once each time you open the app — browsers block sign-in popups that aren't triggered by a click. After that, every change pushes automatically for the rest of the session.
